# Heritage Atlas

A personal UNESCO World Heritage Sites tracker. Browse 190+ sites across 5 regions, mark visits, build a wishlist, journal your experiences, and earn achievement badges.

## Features
- **List view** — Browse, search, filter, and sort all sites
- **Map view** — See all sites plotted on a world map
- **Three-state tracking** — Not visited → Partially visited → Fully visited
- **Wishlist** — Bookmark dream destinations
- **Journal** — Date, rating, and notes for each visited site
- **Badges** — 12 achievements to unlock
- **Persistent** — All data saved in your browser's localStorage

## Setup
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub, then import to [Vercel](https://vercel.com) — it auto-detects Vite.
